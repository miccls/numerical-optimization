\section{Theory of Constrained Optimization}
The formulation of constrained problems in this book,
\begin{equation}
    \underset{x\in \mathbb{R}^n}{\min} f(x) \qquad \text{subject to} \quad 
    \begin{cases}
        c_i(x) &= 0, \quad i\in\mathcal{E},\\
        c_i(x) &\geq 0, \quad i\in\mathcal{I}
    \end{cases}
    \label{eq:constrained-problem}
\end{equation}
is standard in the literature, except for the sign chosen of the inequality constraint functions ($i\in\mathcal{I}$) which may vary from author to author. 
By defining the feasible set as
\begin{equation}
    \Omega \equiv \left\{ x \in \mathbb{R}^n | c_i(x) = 0, \forall i\in\mathcal{E}, c_i(x) \geq 0, \forall i \in \mathcal{I} \right\}
    \label{eq:feasible-set}
\end{equation}
the authors present a very elegant formulation of the constrained problem as
\[
    \underset{x\in\Omega}{\min}f(x).
\]
\subsection{Farkas' lemma}
In the proof of the first order optimality conditions (Theorem 12.1 in the book) the authors use
\begin{Lma}[Farkas]
    Let the cone K be defined as 
    \[
        K = \{By + Cw|y\geq 0\}.
    \]
    Given any vector $g\in\mathbb{R}^n$ we have either that $g\in K$ (i.e. $g = By + Cw$ for some $y\geq 0, w \in \mathbb{R}^n$) or that there exists $d\in\mathbb{R}^n$ satisfying 
    \begin{equation}
        g^T d < 0, \quad B^Td \geq 0, \quad C^T d = 0
        \label{eq:farkas_conditions}
\end{equation}
\end{Lma}
I felt the rest of the proof, after Farkas' lemma was a bit thin so I thought I would expand a bit here.
The proof assumes the constraint qualification LICQ, that the the active constraints $c_i(x^*)$ at the optimum are linearly independent, i.e. the matrix $[\nabla c_i(x^*)]$ of their gradients has full rank.
Then what needs to be done is to show that there exists a vector $\lambda^*$ of Lagrange multipliers which satisfies the KKT conditions.
They have previously showed that for all vectors $d$ in the tangent cone at the optimum $T_\Omega(x^*)$, the following holds
\[
    d^T\nabla f(x^*) \geq 0.
\]
They also show that if the LICQ hold, $T_\Omega(x^*) = \mathcal{F}(x^*)$. Recalling the definition of $\mathcal{F}(x^*)$, the set of first order feasible directions,
\[
    \mathcal{F}(x^*) = \left\{ d \> \left\vert \begin{aligned}
        \quad&d^T  A_\mathcal{I}(x^*) \geq 0\\
        \quad&d^T  A_\mathcal{E}(x^*) = 0
    \end{aligned}\right.\right\}
\]
here $A_\square(x^*)$ is the matrix of either active inequality constraint gradients or equality constraint gradients depending what you replace with \sq.
By identifying $A_\mathcal{I}(x^*)$ with $B$ and $A_\mathcal{E}(x^*)$ with $C$ in Farkas' lemma, we see that there must be some vector $\lambda_i^* \geq 0, i \in \mathcal{A}(x^*) \cap \mathcal{I}$ for which 
\[
    \nabla f(x^*) = A(x^*) \lambda^* = \begin{pmatrix}
        A_\mathcal{I}(x^*)\\
        0
    \end{pmatrix}\lambda^*_\mathcal{I} +  \begin{pmatrix}
        0\\
        A_\mathcal{E}(x^*)
    \end{pmatrix}\lambda^*_\mathcal{E}
\]
since all vectors $d$ that satisfy the latter two of the conditions in \eqref{eq:farkas_conditions} do not satisfy $d^T\nabla f(x^*) < 0$.
The proof is then finished by verifying that this $\lambda^*$, together with the local solution $x^*$ does in fact satisfy all the KKT conditions.

\subsubsection{Exercise 12.22}
We begin by making explicit the implications of the properties of $c_i(x), \i\in\mathcal{E}$ being linear and $-c_i(x), i\in\mathcal{I}$ being convex. 
The defining property of a linear function is the following
\[
    f(ax + by) = af(x) + bf(y)
\]
and for a function to be convex means
\[
    f(\lambda x + (1-\lambda)y) \leq \lambda f(x) + (1 - \lambda) f(y),\qquad \lambda \in [0,1].
\]
In words, the function between two points $x,y$ is bounded above by the straight line connecting $f(x), f(y)$.
Additionally, the negative of a convex function is \textit{concave}, which has the same definition, but with $\leq$ switched for $\geq$.

Now for a set to be convex, we require any convex combination of points within the set to also be members of the set\footnote{A convex combination is a linear combination with non-negative coefficents summing to one. If you allow the coefficents to take any value, but still require them to sum to one, it is called an \textit{affine} combination.}.
That is,
\[
    \lambda x + (1 - \lambda)y = z \overset{!}{\in} \Omega
\]
To show this holds, we need to show $z$ is feasible.
\begin{align*}
    c_i(z) &= 0,\quad i\in\mathcal{E}\\
    c_j(z) &\geq 0, \quad i\in\mathcal{I}    
\end{align*}
Starting with the equality constraints
\begin{align*}
    0 \overset{!}{=} c_i(z) = c_i(\lambda x + (1-\lambda)y) = \lambda c_i(x) + (1-\lambda)c_i(y) = 0
\end{align*}
where the second inequality follows from linearity of $c_i, i\in\mathcal{E}$ and the third from $x,y\in\Omega$.
For the inequality constraints we have
\[
    0 \overset{!}{\leq} c_i(z) = c_i(\lambda x + (1-\lambda)y) \geq \lambda c_i(x) + (1-\lambda)c_i(y) \geq 0
\]
where the second inequality comes from $c_i, i \in \mathcal{I}$ being concave. \hfill\sq
